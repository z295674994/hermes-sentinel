# Smart Money Scanner（智能资金异动扫描系统）

> 基于币安永续合约的资金行为分析系统  
> 不预测涨跌，不预测K线，不预测未来  
> 只识别资金异动，寻找资金已进入但价格尚未充分反映的标的

---

## 一、核心理念

**不做：** 预测BTC/ETH涨跌、预测K线、预测未来价格 → 不可验证、不可持续  
**做：** 寻找资金流入、筹码收集、吸筹行为、趋势启动前兆、板块轮动

**核心问题：** 哪些币种正在被资金关注，但市场尚未形成一致预期？

## 二、系统架构

```
Binance WebSocket / REST
        ↓
  实时数据采集层 (data/)
        ↓
  指标计算引擎 (engine/)
        ↓
  行为模式引擎 (patterns/)
        ↓
  AI分析层 (ai/)
        ↓
  告警推送 (alerts/)
        ↓
  数据库存储 (db/)
        ↓
  历史回测系统 (backtest/)
```

## 三、核心指标

| 指标 | 计算方式 | 意义 |
|------|---------|------|
| OI变化率 | 当前OI / N日前OI - 1 (1d/3d/7d/14d) | 新增资金进入 |
| CVD | 主动买入-主动卖出累计值 | 真实买卖力量 |
| Volume Growth | 近3日均量 / 近30日均量 | 市场关注度 |
| Funding | 变化率+趋势 | 市场情绪 |
| ATR | 波动率 | 是否压缩/扩张 |
| VWAP | 成交量加权均价 | 机构成本参考 |
| VPVR | 成交量分布 | 筹码密集区 |

## 四、行为模式（10种）

| # | 模式 | 条件 |
|---|------|------|
| 1 | 吸筹 | OI↑ CVD↑ 价格小涨 Funding≤0 |
| 2 | 空头回补 | 价格↑ OI↓ |
| 3 | 多头追涨 | 价格↑↑ OI↑↑ Funding↑ |
| 4 | 空头建仓 | 价格↓ OI↑ |
| 5 | 派发 | 价格横盘 成交量↑ CVD↓ |
| 6 | 趋势启动 | 波动率压缩 成交量放大 突破 |
| 7 | 假突破 | 突破后CVD不跟随 OI↓ |
| 8 | 高位换手 | 高位横盘 成交量增加 |
| 9 | 低位换手 | 低位横盘 成交量增加 |
| 10 | 逼空 | 价格暴涨 OI↑ Short Ratio高 |

## 五、Accumulation Score (0-100)

| 指标 | 分值 |
|------|------|
| OI增长 | 30 |
| CVD增长 | 25 |
| 波动率压缩 | 20 |
| 成交量增长 | 15 |
| Funding | 10 |

**分级：** 90+ 重点关注 / 80+ 观察 / 70+ 候选 / <70 忽略

## 六、吸筹效率

公式：`效率 = 资金增长 / 价格涨幅`  
示例：OI +40%, 价格+5% → 效率=8 → 越值得埋伏

## 七、系统输出

每日自动扫描全部USDT永续合约，输出：
- 疑似吸筹榜
- 趋势启动榜
- 逼空榜
- 高位派发榜
- 资金流入榜
- 板块轮动榜
